# Example Client Brief — Lupine Field Coffee

This folder is a **completed reference** showing what a usable client brief looks like.
Read it once before filling in your own `client-brief/` files. The level of specificity
here is what makes Claude Code's suggestions actually useful.

**Fictional client:** Lupine Field Coffee — a single-origin coffee roaster based in
Missoula, Montana. Sells whole-bean coffee, brewing equipment, and a quarterly
subscription. Uses Shopify Basic on the Horizon theme.

This is illustrative only — none of this is a real merchant. Use the structure and
density of detail as a model for your own briefs.

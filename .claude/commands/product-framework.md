description: Design a reusable product description framework from the brief
---

Based on `client-brief/voice-style-guide.md`, `client-brief/business-goals.md`, and `client-brief/page-content.md`, create a product description framework for this brand.

Include:
- Opening hook formula
- Key information hierarchy (what comes first, second, third)
- How to handle variants
- CTA / closing line pattern
- Character/word count targets
- Anti-patterns specific to this brand's voice (things to never write)

Then write 3 example descriptions for plausible products from this catalog (look at `store-data/store-profile.md` for the actual product types). The examples are the proof — the framework is only as good as the descriptions it produces.
